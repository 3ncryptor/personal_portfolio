# Aryan Vibhuti - Professional Portfolio

A professional portfolio website showcasing expertise in ethical hacking, penetration testing, and software development. Built with modern web technologies and advanced animations for optimal user experience.

## Core Features

### User Interface
- Animated hero section with dynamic typing effects and matrix background
- Responsive navigation with smooth scrolling and sticky positioning
- Professional timeline showcasing career progression
- Interactive project showcase with hover animations
- Contact form with client-side validation
- Preloader with custom binary code animation

### Technical Implementation
- GSAP animation library integration for smooth transitions
- ScrollTrigger implementation for scroll-based animations
- Intersection Observer API for performance optimization
- CSS Grid and Flexbox for responsive layouts
- Custom CSS properties for consistent theming
- Mobile-first responsive design approach

### Visual Design
- Matrix rain effect background animation
- Glitch text effects on section headers
- 3D transformation effects on interactive elements
- Gradient overlays and backdrop filters
- Professional color scheme with accessibility compliance

## Technology Stack

### Core Technologies
- HTML5 with semantic markup structure
- CSS3 with advanced custom properties and grid systems
- JavaScript ES6+ for interactive functionality
- GSAP (GreenSock Animation Platform) with ScrollTrigger and TextPlugin

### External Dependencies

## Installation and Setup

### System Requirements
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Local development server (optional)

### Installation Process

Clone the repository:
```bash
git clone https://github.com/3ncryptor/personal_portfolio.git
cd personal_portfolio
```

Development server options:
```bash
# Option 1: Python built-in server
python -m http.server 8000

# Option 2: Node.js http-server
npx http-server

# Option 3: Direct file access
# Open index.html in web browser
```
